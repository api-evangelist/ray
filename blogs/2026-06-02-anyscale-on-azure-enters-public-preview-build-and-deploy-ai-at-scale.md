Anyscale announced the public preview of its AI compute platform on Microsoft Azure, allowing enterprises to run GPU-accelerated workloads on their own infrastructure with native Azure governance and billing integration. The platform supports distributed training, multimodal data processing, and large-scale inference, with customers like Wayve and Xoople using it for autonomous driving and satellite imagery applications. Anyscale consumption draws down from existing Microsoft Azure Consumption Commitments (MACC) and integrates with Azure services including Resource Manager, Entra ID, RBAC, and Cost Management.
